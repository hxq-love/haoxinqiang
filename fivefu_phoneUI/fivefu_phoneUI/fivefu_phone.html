使用方法

                    简单来说引入fivefu_phone.css和require.js就可以了

                    
                    

                    require的入口页面是js/fivefu.js
                    关于require.js的使用方法这里不做具体说明，请自行学习
fivefu_phone.less模块说明
模块名称 是否已集成？ 备注说明
reset.less 是 reset里面基本集成了绝大部分的基础样式，包含颜色、布局、字体大小、列表等等
btn.less 是 按钮的样式集成，依赖reset.css,主要是一些颜色的依赖
take_photo.less 是 拍照组件的一些样式
my_animate.less 是 动画效果，主要用于弹框，toast的显示隐藏动画
other_pages.less 是 一些常用页面的样式集成，主要包括新闻详情、意见反馈、流程、报错页面等等，以后可能还会有所补充
font_family.less 否 这个模块并未集成，因为用到的不太多，并且这里只是提供一些具体的使用方法，不做具体的介绍
swiper.min.css 否 swiper所使用的css并未集成在less中，这个主要是用于幻灯片、图片预览，如果有用到这个部分，请自行引用即可
iconfont.css 否 用到的字体图标在项目中未引用，只是在展示页面、实力页面用到了部分，这里用的是项目【APP规范整理图标】这个项目的图标，这个不同项目建议使用不同的图标
js模块的集成
模块名称 是否已集成？ 备注说明 使用方法
jquery.min.js 是 require(["/js/jquery.min"], function(jq) { 具体内容 });
take_photo.js 是 拍照组件 require(["../js/take_photo"], function(photo) { photo.takePhoto("#photo_1", 3); });
swiper.min.js 是 用于图片放大预览和幻灯片的使用
img_zoom.js 是 图片放大预览 require(["jquery.min", "../js/swiper.min", "../js/img_zoom"], function($, Swiper, Zoom) { Zoom.imgZoom(".img_box"); });
picker.min.js 是 用于下拉滑动选择组件 require(["../js/jquery.min", "../js/picker.min"], function(jq, Picker) { 内容});
popup.js 是 用于弹框和toast的封装 require(["../js/popup"], function(p) { 内容 });
tab.js 是 用于tab菜单切换 require(["../js/tab"], function(tab) { tab.t(); });
fastclick.min.js 是 用于消除手机端300ms延迟的解决 require(["jquery","fastclick"], function(jq,F) { F.attach(document.body );});
